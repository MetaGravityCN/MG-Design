# @mg-design/icons

Vue 3 icon components for MG Design.

```bash
pnpm add vue @mg-design/icons
```

```vue
<script setup lang="ts">
import { House, Search } from '@mg-design/icons'
</script>

<template>
  <House />
  <Search />
</template>
```

Source and issue tracking: [MetaGravityCN/MG-Design](https://github.com/MetaGravityCN/MG-Design).
